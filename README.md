# test_tilda

### Описание тестового задания на позицию QA

Вопрос 1:
Написать чек-лист для проверки формы обратной связи на сайте — есть 4 поля ввода: имя, email, номер телефона и возраст — числовое, которое должно принимать целые значения от 18 до 99.

Вопрос 2:
Описать баг-репорт для блока меню: согласно требованиям меню должно фиксироваться при скролле, но этого не происходит при вашем тесте.

Вопрос 3:
Ваша задача — провести полное тестирование функциональности нашего сайта и подготовить детальные отчеты о найденных ошибках в произвольном виде.
